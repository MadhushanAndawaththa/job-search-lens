Support

Job Hunt Visualizer is maintained as a focused utility for LinkedIn Jobs. This page covers installation, troubleshooting, and feedback paths.

Installation

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Choose Load unpacked and select the project folder.
  4. Open https://www.linkedin.com/jobs/.

Troubleshooting

  • If the popup says Site access is needed, enable extension access for www.linkedin.com and reload the LinkedIn tab.
  • If LinkedIn changes its markup, update to the latest extension build before reporting a bug.
  • If keyword matches are missing, verify the term exists in the job detail pane and that Pause is turned off.

Feedback and issue reports

Report bugs or request features through the repository issue tracker: https://github.com/MadhushanAndawaththa/Job_Search/issues
