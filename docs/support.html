Support

Everything you need to install, troubleshoot, and report bugs.

Getting started

Job Search Lens runs entirely inside Chrome. Highlights work on any page; LinkedIn-specific tools (card dimming + company stats) light up automatically when you visit LinkedIn Jobs. No sign-in required.

  1. Open chrome://extensions and enable Developer mode.
  2. Click Load unpacked and select the project folder.
  3. Navigate to any page you want to scan, or open linkedin.com/jobs for the full LinkedIn workflow.
  4. Open the extension popup, add a keyword, and reload the page if it was already open.

Troubleshooting

  • If the popup shows "Site access needed", enable extension access for the current site (or for all sites) in Chrome's extension settings, then reload the tab.
  • If keyword matches are missing, confirm the term is actually visible on the page and that Pause all is off in the popup.
  • If dimming stops working after a LinkedIn update, check for a newer extension version on GitHub before filing a bug — LinkedIn changes its DOM frequently.
  • If the popup looks broken or the wrong theme flashes briefly, hard-reload the popup (close and reopen). Theme is persisted in localStorage.

Keyboard shortcuts

  • Ctrl/Cmd + Enter inside the keyword input — submit and add the keyword.
  • Escape while a color picker is open — close the popover and return focus to the swatch button.
  • Right-click any selected text on any page — Add "[text]" to Highlighter.

Report a bug or request a feature

Use the GitHub issue tracker. Please include:

  • Chrome version and OS.
  • Extension version (shown in chrome://extensions).
  • The page or URL where the issue happened (or a description if it's private).
  • What you expected vs. what happened.
