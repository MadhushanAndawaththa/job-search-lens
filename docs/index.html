Privacy

Local-only by construction

Your keywords, colors, and settings live in chrome.storage.local on your device. No accounts, no telemetry, no sync, no remote code. The extension is signed and ships only files that are inspectable in chrome://extensions.

0 network calls · 0 analytics SDKs · 0 servers

How it works

Three steps. Then forget it's installed.

Once your keywords are saved, the extension just runs. No dashboards to babysit, no notifications, no daily prompts.

  1. Install

    Load unpacked from GitHub source, or install from the Chrome Web Store once published. No account creation, no email confirmation, no onboarding flow.

  2. Save keywords

    Right-click any selected text and choose Add to Highlighter, or type directly in the popup. Pick a color per keyword.

  3. Scan faster

    Open any job site — your terms light up instantly. On LinkedIn Jobs, processed cards fade and company stats appear inline.
